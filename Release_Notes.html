Purpose

This application implements full LED color control within the Lighting-App from the X-Cube-Matter v1.0.3 package, using the NUCLEO-WB55RG board with external RGB LED bars. For additional information about these components, please consult the official package documentation.

Here is the list of references to the user documents:

  • Matter STMicroelectronics MCU Wiki - ST Wiki dedicated to MCU, Matter page
  • DB5144 - Data Brief of the Matter software expansion for STM32Cube

LightingColorControl-App example is being delivered as-is, not necessarily supported by ST.

Update History

Main changes

  • Lighting Color Control application without OTA/SBSFU

Here is the list of provided features:

  • Transforming HSV (Hue, Saturation and Value) into RGB with level control.
  • Transforming CT (Color Temperature) into RGB with level control.
  • Transforming XYZ color space into RGB with level control.
